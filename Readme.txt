 Fake Review Detection project:

Goal: Detection of the fake reviewers and futher work on detecting the products buying these fake reviews.
Method: Instead of focusing on the reviews and sentiment analysis using NLP methods, developed a different approach using the Graph Network Analysis. 
Data: Collected data from actual social media networks which serve as the connection between fake reviewers and product owners.
Process: Data collection, organization, cleaning and network analysis. Other ML algorithms are also tested for verification.
